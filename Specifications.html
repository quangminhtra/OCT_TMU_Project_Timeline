Hardware Components

  • Core: RAK3172 (LoRaWAN) & L80-R (GPS)
  • Power: LiPo 604030 Battery + MCP1700 Regulator
  • Housing: Custom 3D Printed Resin (Waterproof)
  • Base Station: Raspberry Pi Pico 2 W
  • Connectivity: 915 MHz Ceramic Chip Antenna

Software Ecosystem

  • Frontend: React Native Mobile App
  • Backend: Firebase (Real-time DB & Auth)
  • Features: Live GPS Map & Geofencing
  • Design: Prototyped in Figma (Link to Figma).

Key Metrics

Weight: ~25.2g (Target < 30g)

Range: >1 km

Cost: ~$40.57 CAD
