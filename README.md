# OCT TMU Project Timeline

## Project Purpose
The OCT TMU Project Timeline aims to provide a comprehensive overview of the project lifecycle, tracking key milestones, tasks, and timelines to ensure efficient project management. This tool is essential for teams aiming to enhance collaboration, improve transparency, and ensure timely delivery of project objectives.

## Features
- **Visual Timeline:** An intuitive interface that displays the project timeline at a glance, making it easy to spot upcoming tasks and deadlines.
- **Milestone Tracking:** Set, edit, and view important project milestones to ensure alignment with project goals.
- **Task Management:** Create, assign, and monitor tasks within the project to keep all team members accountable and informed.
- **Collaboration Tools:** Facilitate discussions and share updates among team members directly within the timeline interface.

## Technical Details
- **Built With:** The project uses modern web technologies including HTML, CSS, and JavaScript for the frontend, and Node.js for the backend.
- **Database:** MongoDB is utilized for storing project data, ensuring scalability and flexibility.
- **Version Control:** Managed through GitHub to track changes and collaborate effectively across the development team.

## Installation Instructions
To get started with the OCT TMU Project Timeline:
1. Clone the repository: `git clone https://github.com/quangminhtra/OCT_TMU_Project_Timeline`
2. Install dependencies: Navigate to the project directory and run `npm install`.
3. Start the application: Run `npm start` to launch the project locally.
